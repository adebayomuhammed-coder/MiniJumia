# MiniJumia

MiniJumia is a Flask-based web application inspired by modern e-commerce and authentication systems. The project includes user registration, login authentication, and PostgreSQL database integration.

## Features
- User registration and login
- Authentication system
- PostgreSQL database support
- Environment variable protection using `.env`
- Hosted on Render
- Flask backend integration

## Technologies Used
- Python
- Flask
- PostgreSQL
- SQLAlchemy
- Gunicorn
- HTML/CSS
- Render
- simple js 

## Notes
This project is currently configured with a personal email setup for authentication and testing purposes.
